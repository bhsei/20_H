### 核心功能
* 内置多种包含主流网站数据源的模板，如京东、天猫、大众点评等热门采集网站，模板提供设置参数即采集的字段
* 管理采集任务，包括添加采集任务、查看采集的实时进度、启动采集、暂停采集、终止采集
* 管理采集结果，包括浏览采集结果、保存采集结果到本地
* 使用Scrapyd管理爬虫任务，实现分布式爬虫
* 使用Selenium实现应对反爬虫机制
* 使用统一的代理池实现应对反爬虫机制
